Connect your GitHub account

This admin panel saves your products directly to GitHub so the live site updates automatically. You only need to do this setup once per device.

How to get your token:

  1. Go to github.com/settings/tokens/new
  2. Give it any name (e.g. wornsum-admin)
  3. Select the repo scope checkbox
  4. Click Generate token and copy it

Stored locally on this device only. Never sent anywhere except GitHub's API.
